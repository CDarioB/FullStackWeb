Lista Oredenada.

<< ol [type="1"] [start="20"] >>

[type="1"] (opcional) ... Nos permite cambiar el tipo de viñeta de la lista.. Su valor puede ser Numérico (1); Alfabético (A) ; Numérica romana (I)

[start="20"] (opcional) = Nos permite definir dónde va a empezar nuestra numeración. Pueden ser valores negativos

Ejemplo de lista ordenada:
  1. Primer ítem
  2. Segundo ítem
  3. Tercer ítem

Lista Desordenada.

<< ul [type="disc"] >>

[type="disc"] (opcional) ... Nos permite cambiar el tipo de viñeta de la lista .. Su valor puede ser Disc (●) ; Circle (○) ; Square (◼) ; None

Ejemplo de lista desordenada:
  • Primer ítem
  • Segundo ítem
  • Tercer ítem
Listas Anidadas.
  • Dentro de un ol o un ul solo puede haber elementos li.
  • Las listas se pueden anidar agregando un ol o ul dentro de un li.

  • Recordar para el viaje:
